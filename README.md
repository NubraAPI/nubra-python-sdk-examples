# Nubra API Examples

This repository contains runnable examples for using the Nubra Trading
and Market Data APIs.

## Examples Included

### Python
- Authentication
- Place / modify / cancel orders
- Market data (LTP, OHLC)
- Margin calculation

### cURL
- Raw REST API examples

## Prerequisites
- Python 3.9+
- Nubra API credentials
